# Novel2Play

## 项目背景
此项目为七牛云暑期实训云比赛项目，该期共三个主题：

题目一：AI 英语口语陪练

题目二：AI 同声传译助手

题目三：AI 小说转剧本工具

本项目为题目三，选择题目三的原因有二：

1.另外两个需要语音识别，语音识别的AI不算便宜；

2.最近才买了deepseek的token，便宜好用；

## ✨ 主要功能

- ✅ 自动提取 **角色**（含外貌、性格、身份简介）
- ✅ 自动划分 **场景**（地点、时间、氛围、视觉提示）
- ✅ 将对话转换为 **台词**，附带动作与情绪描述
- ✅ 将叙述性文字转换为 **动作** 或必要的 **旁白**
- ✅ 输出 **YAML** 格式剧本，便于二次编辑和解析
- ✅ 支持 **长篇小说**（自动分块处理，保持角色与情节一致）
- ✅ 利用 **DeepSeek API** 高质量生成，成本极低

## 🧠 技术架构

- **语言**: Python 3.8+
- **核心依赖**:
  - `requests` – 调用 DeepSeek API
  - `pyyaml` – 生成 YAML 剧本文件
  - `python-dotenv` – 管理 API Key 环境变量
- **AI 模型**: DeepSeek-V4-Flash / DeepSeek-Chat（可配置）
- **处理策略**: 按段落边界分块（默认 4000 字符/块），顺序调用 API，维护上下文状态（角色列表、场景编号、剧情摘要），确保长文本转换的连贯性。

## 🚀 使用指南
1.下载并解压该项目；

2.运行exe文件；

3.在网页端输入自己的API Key，并上传小说文本文件；

4.点击运行，生成剧本
